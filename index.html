Welcome to ClickColor!

This is a fun puzzle game where you must burn some neurons to equalize all the colors on the board!

The rules are simple:

  1. When you click on a square the square's color changes and the color of his adjacent squares change too.
  2. To win you must change all the squares so they have the same color.

ClickColor was started by @runys to improve his programming skills with C++ and the SDL library. Later, at a game programming gathering, he brought the game and suggested to use it as the subject of a game coding dojo.

Then @leonn, @mfurquim and @athos-ribeiro got involved and it was decided to open the code to the world, so anyone could see our work and bring their suggestions and ideas into it.

So ClickColor is an open-source C++/SDL game under GPL license that anyone can use to study or improve.

Show us your love

You can easily start to contribute by forking the repository here and running:

  $ git clone git@github.com:your_account/ClickColor.git
            $ git remote add oficial git@github.com:runys/ClickColor.git
            $ git pull oficial master

All you need to develop is:

  • C++ compiler (g++)
  • SDL library
  • SDL_ttf
  • SDL_img
  • SDL_mixer

There is some neat tutorials about configuring SDL at LazyFoo

If you are an artist and want to work with us, we use:

  • Adobe Photoshop CS6
  • Graphics Gale 1.93.21
